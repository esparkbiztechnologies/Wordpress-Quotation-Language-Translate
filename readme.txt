== Description ==

  This plugin is used for language translation.

- It is compatible for any WordPress site.
- Lists out orders from customer.
- Manage order, language selection, option section and thank page from back end.
- Also order, language selection, option section and thank page templates can be overwritten from current theme.
- Also manage email template for administrator and customer as well as manage exchange rate setting from back end. 
- Customer can put content as well as content file for language translation.
- Then after select language which you want to convert it and manage content which you entered.
- Then after manage simple, premium and business plan option.
- Also convert price with your currency.
- Also moving to previous page using by right side widget.

== Installation ==

1. Upload the entire language-translate folder to the /wp-content/plugins/ directory, or just upload the ZIP package via 'Plugins > Add New > Upload' in your WP Admin
2. Activate ESB Language Translation from the 'Plugins' page in WordPress
